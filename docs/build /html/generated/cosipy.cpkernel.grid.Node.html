cosipy.cpkernel.grid.Node¶

class cosipy.cpkernel.grid.Node(height, snow_density, temperature, liquid_water_content, ice_fraction=None)¶

Bases: object

The Node-class stores the state variables of layer.

The numerical grid consists of a list of nodes that store the information of individual layers. The class provides various setter/getter functions to read or overwrite the state of individual layers.

Parameters

  • height (float) – Height of the layer [\(m\)]

  • density (float) – snow density [\(kg~m^{-3}\)]

  • temperature (float) – temperature of the layer [\(K\)]

  • liquid_water (float) – liquid water [\(m~w.e.\)]
